# HouseSwipe

A React + Vite prototype for a Tinder-style house swiper app.

## Included
- dark theme mobile UI
- Rightmove / OnTheMarket provider toggle
- integrated mobile filters
- map embed per property
- floor-plan insight placeholder section
- local price-per-square-foot comparison card
- shortlist / pass interactions

## Run locally

```bash
npm install
npm run dev
```

## Open in Visual Studio
Open the extracted folder in Visual Studio or Visual Studio Code, then run the commands above in the terminal.

## Important note
The price checker currently uses placeholder benchmark data in code so the project is self-contained. It is not performing a live lookup against Housemetric yet.
